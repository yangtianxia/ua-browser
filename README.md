# uaBrowser - 浏览器检测

通过userAgent和浏览器环境变量检测浏览器、系统及设备类型的常用工具

## 安装

### npm

```sh
npm i ua-browser
```

### pnpm

```sh
pnpm add ua-browser
```

### yarn

```sh
yarn add ua-browser
```

## 支持方法

```typescript
import { uaBrowser } from 'ua-browser'

/** 浏览器解析 */
uaBrowser(ua?: string): {
  version: string;
  osVersion: string;
  engine: string;
  browser: string;
  os: string;
  device: string;
  isWebview: boolean;
  language: string;
  platfrom: string;
}

/** 检查 `webview` 浏览环境，仅支持 `android` */
uaBrowser.isWebview(ua: string): boolean

/** 检查微信小程序 */
uaBrowser.isWechatMiniapp(): boolean

/** 浏览器语言 */
uaBrowser.getLanguage(): string

/** 当前版本 */
uaBrowser.VERSION: string
```
